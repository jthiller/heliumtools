# Helium Tools Pages

This directory contains the frontend code for heliumtools.org, built with Vite and React.

## Setup

1.  Install dependencies:
    ```bash
    cd public
    npm install
    ```

## Development

To run the frontend locally:

```bash
cd public
npm run dev
```

This will start the development server on `http://localhost:5173`.

## Building

To build for production:

```bash
cd public
npm run build
```

The build output will be in `public/dist`.
